# Green Sun Online

## 🎨 About the Project
**Green Sun Online** is a digital showcase website designed to display the artisanal works, paintings, and crafts of an artist. Currently acting as a visual portfolio and gallery, the project is built with a scalable, modern architecture to support a seamless transition into a fully-featured e-commerce platform in the future.

## 💻 Tech Stack
This repository is organized as a monorepo, separating the user interface from the content management system.

* **Frontend:** [Next.js](https://nextjs.org/) (React framework) - Located in the `/frontend` directory.
* **Backend:** [Strapi](https://strapi.io/) (Headless CMS) - Located in the `/backend` directory.

## 📂 Project Structure
- `/backend`: Contains the Strapi CMS for managing and storing data.
- `/frontend`: Contains the Next.js application responsible for the user interface and connecting to the Strapi API.
